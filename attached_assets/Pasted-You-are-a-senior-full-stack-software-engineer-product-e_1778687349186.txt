========================
BUSINESS RULES
========================

Implement these rules:

1. Only lecturers and admins can create or edit questions.
2. Students can access only their enrolled courses.
3. Archived questions cannot be used in newly generated exams.
4. Only active/approved/demo-approved questions can be used in exams.
5. Answer randomization must preserve grading correctness.
6. Every generated mock exam must be stored.
7. Every submitted mock exam must store selected answers, correctness, score, and status.
8. Unauthorized users must receive 403 Forbidden.
9. Unauthenticated users must receive 401 Unauthorized.
10. Input must be validated.
